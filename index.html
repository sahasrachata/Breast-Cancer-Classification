Building a Convolutional Neural Network for Breast Cancer Classification¶

The provided code is a TensorFlow-based implementation of a Convolutional Neural Network (CNN) using the Keras API for binary classification on the breast cancer dataset. The dataset is loaded using scikit-learn's datasets module, and relevant preprocessing steps, such as standardization and train-test split, are performed. The CNN model consists of convolutional layers with batch normalization and dropout for regularization. The model is compiled using the Adam optimizer and binary crossentropy loss function. Training is executed with 35 epochs, and the learning curve, displaying accuracy and loss over epochs for both training and validation sets, is visualized. The code also includes comments explaining key components and imports, enhancing readability and providing insights into the purpose of each section.

# Creating a Sequential model
model = Sequential() # Initialize a Sequential model
# Convolutional layer with 64 filters and kernel size of 3
model.add(Conv1D(filters=64, kernel_size=3, activation='relu', input_shape=(X_train.shape[1], 1)))
model.add(BatchNormalization())
model.add(Dropout(0.5)) # Introducing dropout for regularization

# Second convolutional layer with 128 filters and kernel size of 3
model.add(Conv1D(filters=128, kernel_size=3, activation='relu'))
model.add(BatchNormalization())
model.add(Dropout(0.5)) # More dropout for regularization

# Flatten layer to transition from convolutional layers to dense layers
model.add(Flatten())

# Dense layer with 128 units and ReLU activation, also using L2 regularization
model.add(Dense(128, activation='relu', kernel_regularizer=l2(0.01)))
model.add(Dropout(0.5)) # Adding dropout for regularization

# Compile the model with a learning rate scheduler
def lr_schedule(epoch):
    lr = 0.001
    if epoch > 10:
        lr *= 0.1
    return lr

# Plot the learning curve
def plot_learning_curve(history, epochs):
    epoch_range = range(1, epochs + 1)
    
 # Plotting the accuracy curve for training and validation data   
    plt.plot(epoch_range, history.history['accuracy'])
    plt.plot(epoch_range, history.history['val_accuracy'])
    plt.title('Model Accuracy')
    plt.xlabel('Epoch')
    plt.ylabel('Accuracy')
    plt.legend(['Train', 'Validation'], loc='upper left')
    plt.ylim(0, 1.1)
    plt.show()

 # Plotting the loss curve for training and validation data   
    plt.plot(epoch_range, history.history['loss'])
    plt.plot(epoch_range, history.history['val_loss'])
    plt.title('Model Loss')
    plt.xlabel('Epoch')
    plt.ylabel('Loss')
    plt.legend(['Train', 'Validation'], loc='upper left')
    plt.show()
